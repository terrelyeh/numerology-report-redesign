## Tech Stack

- **Tailwind CSS** (CDN) — utility-first styling
- **Google Fonts** — Instrument Serif + DM Sans
- **Vanilla JS** — canvas particles, orbit animation, form handling
- **GitHub Pages** — auto-deploy via Actions

## Design System

| Token | Value | Usage |
|-------|-------|-------|
| Cream | `#EDE8E0` | Light section backgrounds |
| Warmgray | `#2C3345` | Body text (never pure black) |
| Sage | `#3A6B5E` | V1 accent, links |
| Teal | `#1A6B6A` | V2/V3 primary accent |
| Teallight | `#2A9D8F` | CTA buttons, highlights |
| Gold | `#B8964E` | Premium features, stars |

## Local Development

Just open any `.html` file in a browser. No build step required.

```bash
# Or use any static server
npx serve .
```
